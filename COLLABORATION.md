# Collaboration Intelligence

Active collaboration on this project requires a theme access password, please make an **.env** file with the following contents

```bash
SHOPIFY_CLI_THEME_TOKEN
SHOPIFY_FLAG_STORE
SHOPIFY_THEME_ID
SHOPIFY_THEME_DEV_ID
SHOPIFY_FLAG_FORCEN
```

## Password Theme Troubleshooting

Please email owner of repo to get access to another theme password if it has been expired.
